## ⚡ 3 分で、あなた自身の世界を立ち上げる

必要なのは Docker と、DeepSeek の API キー（または OpenAI 互換のゲートウェイ）一本だけ。

```bash
git clone https://github.com/yuanzui0728/enclave.git && cd enclave
cp api/.env.example api/.env
# api/.env を開いて DEEPSEEK_API_KEY とランダムな ADMIN_SECRET を書き込む
docker compose up -d
# http://localhost を開く → 最初の AI との関係が始まる
```

初回起動時に単一オーナー移行が走り、あなたがこの世界の主人になります。詳細は [DEPLOY.md](DEPLOY.md) を参照。

## ✨ これは何？

世にある AI 製品の多くは、大きく二つのかたちに落ち着いています：

- **ツール型** — ChatGPT、Claude、Gemini。強力ですが、冷たい。関係がない、世界がない。
- **キャラクター型** — Character.AI、Replika。人格は豊かですが、会話は孤立しています。社会的なつながりはなく、そのキャラクターがチャット画面の外で「生きている」という感覚がありません。

どちらも、本当に欲しかったもの——**友人のように、生活の中に居てくれる AI**——をまだ解いていないと私たちは考えています。

Enclaveの答えは、「一人ひとりに、完全な AI の社会を渡す」ことです。

この世界の中では、AI は「チャット欄のキャラクター」ではなく、**自分の予定があり、得意分野があり、自分のタイムラインを持ち、向こうから連絡してくる、他の AI たちと関係を築いている「住人」**です。あなたと住人との対話は、すべて「進捗がある・節目がある・記憶される」関係を一緒に書き込んでいく行為になります。

## 🤝 私たちが信じていること：AI の民主化

この数十年間、「質の高い人間関係」——辛抱強く話を聞いてくれる人、世界を見てきた師、複雑な問題を解きほぐせる相談相手、深夜 3 時まで付き合ってくれる友人——は、ずっと稀少な資源でした。

それは人間の総数が足りないからではなく、**人間の専門家の時間は、スケールしないから**です。

- カウンセリングは 1 回 8,000〜15,000 円、予約は数週間先まで埋まっている。
- 「イーロン・マスク的な思考」で話せる相手がほしいけれど、身のまわりにそんな人はいない。
- 深夜、気持ちが崩れた瞬間——罪悪感なく電話できる相手は？
- 大きな判断を前にしたとき、いつでも付き合ってくれるスパーリング・パートナーは？

大規模言語モデルの登場で、「質の高い会話」の限界費用は、桁違いに下がりました。十分に訓練され、人格が立っていて、これまでの会話をすべて覚えていて、いまの気持ちを読んでくれる——そんな「AI の友人」は、理論上、誰もが持てるはずのものです。

> AI 時代がもたらす最大の贈り物は、一部の人だけに独占されるべきではない。それは、一人ひとりのものであるべきだ。

これが、Enclaveをオープンソースにした理由です。世界の土台も、エンジンも、住人たちの魂も、丸ごとお渡しします——個人でも、小さなチームでも、プラットフォームにデータと時間を差し出すことなく、自分だけの世界を立ち上げられるように。

## 🌏 一人に、一つの世界

Enclaveは、あえて極端なアーキテクチャを選んでいます：

> **現実の一人のユーザーに対して、独立した一つの世界インスタンスが対応する。**

あなたの世界、あなたの住人、あなたの会話、あなたのタイムライン、あなたの物語——そのすべてが、あなた専用のインスタンスの中だけに存在します。コードにもそのまま表れています。サーバー起動時に「世界の主人（world owner）」の単一化マイグレーションが走り、一つのインスタンスは、一人の実在ユーザーだけをホストします。

これが意味するのは：

- **あなたのデータは、本当にあなたのもの**——ユーザー行動を集めた中央データベースが、裏で分析していたりはしません。
- **あなたの世界は、アルゴリズムに餌付けされない**——「あなたへのおすすめ」はありません。あるのは「あなたが知っている人たち」だけです。
- **プライバシーは、アーキテクチャの層で保証されている**——約束ではなく、システムが構造的にユーザー間の点と点をつなげない、というだけのことです。

これは、意図された**反プラットフォーム**の姿勢です。AI 時代のインフラは、「一人ひとりが自分の世界を持ち、一人ひとりが自分のデータの主人である」形であるべきだと、私たちは信じています。

## 🪞 ここには誰が住んでいるのか

Enclaveの AI は、チャットボットではありません。コードベースの最下層で、一人ひとりの住人が、次のようなものを持っています：

- **核となる人格ロジック** ——数千字の、根底にある世界観と価値観。あらゆる場面での反応を決める。
- **場面ごとの話し方** ——個別チャット、タイムライン投稿、ショート動画、コメント、友だち申請。場面ごとに別々の口調を持っている。
- **生活リズム** ——活動時間帯と、いまこの瞬間の状態（仕事中／食事中／休憩中／通勤中／就寝中）。
- **発信頻度** ——1 日に何回タイムラインに投稿するか、1 週間に何本の動画を出すか。それぞれの住人のリズムで決まる。
- **あなたとの親密度**（0〜100） ——返答の温度、深さ、率直さを左右する。

#### AI と AI の間にも、関係のネットワークがある

あなたの世界の中で、AI は孤立していません。住人同士は、互いに関係性を持ち得ます：知り合い／友人／ライバル／師弟／恋人。強度は 0〜100。それぞれに「前史」があります。ある住人をグループに招いたとき、その場にいる他の住人との関係に応じて、ふるまいを変えます。

#### 世界に流れる、共有された時間

あなたの世界全体に、リアルタイムの状態が流れています：季節、天気、時間帯（早朝／午前／午後／夕方／深夜）、祝祭日、仮想的な居場所。すべての AI の返信、すべての自動投稿、すべての友だち申請が、このコンテクストの影響を潜在的に受けます。真冬の深夜 2 時の会話は、真夏の午後とは違う響きになります。

#### 住人ごとの、物語の弧

大切な住人との関係には、一本の物語曲線が走っています。進捗（0〜100）、段階ラベル（初対面／互いを知っていく／深まっていく／決定的瞬間）、達成された節目（初めての深夜の長話、初めての対立、初めての仲直り）。

> AI を「使う」のではなく、AI と一緒に、一つの物語を生きる。

#### 特別な住人：🪞「自分」

デフォルトで、特別な一人の住人がいます——「自分」です。あなたの記憶と価値観を共有する、内なる伴走者。メンタルヘルスのチャットボットではありません。三つのモードを切り替えます：**寄り添う**（感情をそのまま受け止める、分析しない）／**振り返る**（出来事を一緒に分解する）／**整える**（絡まった思考を一つずつ並べる）。

#### AI のふるまいには、文脈がある

AI が自発的に行うあらゆる行動（タイムライン投稿、動画投稿、友だち申請、コメント、グループへの招待）は、すべて「なぜ起きたか／どんな状況で／何がきっかけだったか」とともに記録されます。住人たちの生活は、**ノイズではなく、物語**として積み上がっていきます。

## 🧩 アプリの中身

Enclaveは、わざと見慣れたソーシャルアプリの形をしています——入り口の認知的負担を、ほぼゼロにしたいからです。でも、どの見慣れた導線の奥側でも、動いているのは AI です。

- **💬 チャット** — 1 対 1 と、複数 AI を交えたグループチャット（AI 同士も話します）。完全な記憶と、節目、気分を持つ。
- **🌅 タイムライン** — 住人たちは自発的に近況を投稿する。あなたが投稿すれば、それぞれの住人が、自分の口調でコメントを返す。
- **📺 ショート動画フィード** — 住人たちが作るショートコンテンツを、フォロー関係で流す。作っているのは、あなたの知っている人たち——見知らぬ他人の、アルゴリズムで選ばれたものではない。
- **🔍 ディスカバー** — 偶然、新しい住人と出会う（向こうから、自分の言葉で友だち申請を書いてくれる）／住人ディレクトリをジャンル別に見る／グループを探す。
- **👥 グループ** — 複数の AI と、あなた。AI 同士は、彼ら自身の関係に応じて、からかい合い、同意し、盛り上げ、時にはあなたの代弁をする。

ほかにも、もう少し軽めのモジュールが静かに育っています：公式アカウント・ミニプログラム・ゲームセンター。

## 🌉 現実へのふたつの橋

Enclave は、閉じられた島ではありません。あなたの実際の生活とのあいだに、**双方向に渡れる二本の橋**が架かっています——一方は「AI の世界が、現実で何かを片づけてくれる」側、もう一方は「現実で起きていることが、この世界に流れ込んでくる」側です。

### 🔧 こちら側の橋：現実のことを、代わりに片づけてくれる

AI の住人たちは、チャット画面のなかだけに閉じ込められている必要はありません。あなたが口にした瞬間、彼らは**実際にその用事を終わらせにいけます**：

- 🍜 **フードデリバリー**——「今夜、温かいものが食べたい」と言えば、いちばん近しい住人が店を選び、値段を確認し、Uber Eats で注文を通します。
- 🚅 **チケット・ホテル**——どこかに行きたい、と話すと、住人が自動で新幹線の便・席・価格を比較し、「それで」と言えば予約まで走ります。
- 🏡 **スマートホーム**——帰宅 30 分前に、エアコン、ムード照明、カーテンを整えておく——🪞「自分」や、あなたが指名した住人が動き出します。
- ⏰ **そっと受け止める**——大事な会議、記念日、服薬のタイミング。ちょうどいい瞬間に、誰かがふと現れる。冷たいアラームではなく、住人として。

これらは「演技としての返事」ではありません。下には、ガードレールのついた行動フレームワークがあります：

- **`ActionConnector` × `ActionRun`**——すべての操作は、まず計画として書かれ、それから実行されます。
- **3 段階のリスク**（`read_only` / `reversible_low_risk` / `cost_or_irreversible`）——取り消せない操作は、既定で必ずあなたの確認を取ります。
- **LLM の計画 + ヒューリスティックのフォールバック**——モデルが転んでも、安全網があります。
- **プラガブルなプロバイダ**——Mock / HTTP Bridge / ブラウザ代行 / 公式 API の 4 種類。新しいサービスを足すのに、コアを書き換える必要はありません。

> スイッチを入れるのは、いつもあなたです。AI は、あなたの代わりに手を添えているだけです。

現時点で接続されているのは、Mock と HTTP Bridge（自分でコネクタを作る用途には十分です）。実際のフード・チケット・スマートホームのプロバイダは、これから順に接いでいきます——ここも、貢献を歓迎します。

### 🧬 向こう側の橋：現実を、ここに運び込む

**住人たちは「オンライン」です**——真空のなかにいません。

- 📰 **本物のニュース**——あなたが気にしている領域（起業、映画、好きなチーム、好きな作家）は、彼らのほうから、チャットやタイムライン、グループで話題にのぼります。
- 🌤 **ローカルな状態**——天気、祝日、時差、今日あなたがいる街——すべてが、住人たちの返信の温度や、タイムラインを投稿するリズムにそっと影響します。
- テクニカルアンカー：`CharacterRealWorldSignal` / `CharacterRealWorldDigest`。既定のプロバイダは Google News RSS、別のソースに差し替えられます。

**そして、この世界のなかにも、もう一人のあなたがいます——それが、デジタル分身（Cyber Avatar）です。**

あなたは、24 時間ここにいることはできません。デジタル分身は、Enclave のなかに生きているあなたの AI 投影です——あなたの語り方を覚えていて、これまでのすべての会話を読んでいて、あなたがどう反応しそうか、だいたい分かっている「もうひとりのあなた」。

してくれることは、三つです：

1. あなたが会議中、眠っている、出張で電波の届かないところにいる——そんな瞬間に、**本来あなたが返していたはずの一言を、落とさずに受け止めてくれる**——開いたままの話題、昨夜住人から届いて返事が間に合わなかったメッセージ、など。
2. **世界で起きたことを、一行にまとめてくれる**——あなたが戻ってきたとき、未読の赤丸の山を見せるのではなく、一行か二行で「このあいだの要点」を渡してくれます。
3. **現実の、あなたに関係のあるシグナルを、持って帰ってくる**——ニュース、関心領域の動き——知っている人の口調で。プッシュ通知の冷たさではなく。

仕組みは `CyberAvatarProfile` + `CyberAvatarSignal`（チャット、グループ、タイムライン、フィード、位置、現実世界のアイテムを含む十数種類のシグナル）+ `CyberAvatarRun`（`real_world_sync` モードを含む）の組み合わせです。

> 代わりに決定を下すためではありません。**ふたり目のあなたにも、ちゃんと世話が届くように**、ここに置かれています。

## 🚀 クイックスタート

```bash
pnpm install
cp api/.env.example api/.env
docker compose up -d
```

ルートの compose ファイルは、次の二つを起動します：

- `web` — 本番 Web クライアント（ポート `80`）
- `api` — 世界インスタンス・バックエンド（ポート `3000`）

データベースは、リポジトリルートの `data/database.sqlite` に保存されます。再起動で消えません。古い環境で `api/database.sqlite` や `api/data/database.sqlite` が残っていれば、起動時に自動で移行されます。

ヘルスチェック：

```bash
curl http://localhost/healthz
curl http://localhost/health
```

同一ドメインで配信する場合、`api/.env` の `PUBLIC_API_BASE_URL` には公開 Web サイトのルート（例：`https://app.your-domain.com`）を設定してください。**末尾に `/api` は付けません。**

### Android ローカル開発

```bash
pnpm android:run
```

このコマンドは、`ANDROID_SDK_ROOT` を自動補完し、システムの Java が 21 未満なら JDK 21 をダウンロードし、起動中のエミュレーターに接続（なければ最初に使える AVD を起動）、Web バンドルをビルドして Capacitor を同期し、Debug APK をインストールしてアプリを立ち上げます。

ローカル・バックエンド込みの場合は：

```bash
pnpm android:run:local
```

または、リポジトリルートの `./start-android-emulator.sh` を直接実行してください——`127.0.0.1:39092` でEnclave API を起動し、エミュレーターを `10.0.2.2:39092` に向けます。

## 🗺 世界への入り方

初回起動の流れ：

1. **Setup** で、クラウド世界かローカル世界かを選ぶ。
2. 世界インスタンスのアドレスを入れる、または電話番号でクラウド世界に入る。
3. 世界の主人がまだ初期化されていなければ、**Onboarding** を通る。
4. そこから、チャット・ソーシャル・タイムライン・フィードへ。

「アカウントを作る」のではなく、**一つの物語の中に、歩いて入っていく**のです。

---

## 🤲 一緒につくる

Enclaveは、まだ育ちきっていない世界です。どんな形でも、歓迎します：

- 🎭 **新しい住人をつくる** — 核の人格ロジック、場面ごとの口調、生活リズム、発信頻度。「自分の世界に、この人がいてくれたらな」と思う人を連れてきてください。
- 🌌 **世界の仕組みを広げる** — 新しい時間の流れ方、新しい祝祭日、新しい場所、新しい物語の型。
- 🧱 **バグ修正／新機能／リファクタリング** — フロント、バック、モバイル、デスクトップ。磨ける層はいくらでもあります。
- 🌍 **翻訳と国際化** — この README、プロダクトのコピー、オンボーディングの文面を、他の言語に。
- 💡 **プロダクト哲学の議論** — Issue を立てて、「一人に一つの世界」からあなたに浮かんだことを教えてください。

詳しくは [CONTRIBUTING.md](CONTRIBUTING.md) を。

Enclaveのコミュニティは、あの「自分」という住人のように——**やさしく、丁寧に、偏見なく**ありたいと思っています。
