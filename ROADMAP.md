# 路线图 / Roadmap

隐界是一个还在长大的世界。这份路线图是"我们打算往哪里走"的一张大致地图 —— 它不是合同，具体节奏会根据贡献者节奏和真实用户反馈调整。

如果你对其中某一项特别感兴趣、想贡献 / 想先用上 / 想一起讨论，欢迎去 [Discussions](https://github.com/yuanzui0728/enclave/discussions) 开一帖，或直接在对应的 Issue 下留言。

---

## ✅ v0.1.x（当前版本：首次公开 → 稳定化）

> 重点：把"首次公开版本"打磨到路人可以跑起来、玩得起来。

- [x] 一键 Docker 部署（3 分钟从 git clone 到能用）
- [x] 多平台联系人导入链路（微信 4.x、WeFlow、ChatLab JSON/JSONL）
- [x] "自己"角色：陪伴 / 复盘 / 整理 三档
- [x] 赛博分身（Cyber Avatar）行为闭环
- [x] 多模态聊天：图像 / 音频 / 文档（含 OCR / PDF 兜底）
- [x] 提醒任务系统（自然语言 + 管理后台编排）
- [x] 真实世界同步：Google News RSS → 角色聊天/朋友圈
- [x] 多模型推理路由 + 模型人格绑定
- [x] 四语 README（中 / 英 / 日 / 韩）+ 全端 i18n
- [x] Issue / PR / 用户反馈模板 + 微信社群
- [ ] 在线体验 demo 换 HTTPS 域名（IP → 正式域名）
- [ ] 核心功能 3–4 秒 demo GIF 与截图 gallery
- [ ] README 多语言翻译校对（参见 #5）

## 🔜 v0.2（下一站：更像"一个活着的世界"）

> 重点：角色生活更有密度，世界更有呼吸感。

- [ ] **角色模板库 v1**：把现有角色 + 社区贡献的角色整理成可一键引入的模板
- [ ] **朋友圈表情回应**：除了评论，支持 emoji reactions
- [ ] **群聊节奏层**：AI 与 AI 之间真正的"话轮切换" + "起哄/附和"概率模型
- [ ] **Onboarding 可跳过 / 可回退**：5 幕叙事不再是单向一条路
- [ ] **视频号内容池扩充**：更多类型（Vlog / 短剧 / 笔记）、更贴近真实推荐体验
- [ ] **世界时间事件**：节假日、节气、城市事件自动触发对应叙事
- [ ] **首批真实 Provider**：外卖 / 票务 / 智能家居 —— 从 Mock 切到至少一个可用的真实接入
- [ ] **性能**：冷启动时间 -30%，移动端 bundle 减肥

## 🗺 v0.3（中期：让世界"沉下来"）

- [ ] **长程记忆层**：不再只靠对话上下文，把角色对你的记忆沉淀成可检索、可反省的档案
- [ ] **多世界切换**：一个实例可以容纳多套世界观（比如日常 / 创业 / 学习 / 亲密），而不是一锅烩
- [ ] **角色之间的"新闻"**：AI 居民主动在彼此间发生故事，你从朋友圈 / 群里看到而不是被告知
- [ ] **可导出的"世界快照"**：你的世界可以打包带走、换机恢复、分享给朋友
- [ ] **移动端原生壳打磨**：iOS / Android 贴近 App Store 上架级别
- [ ] **插件体系**：第三方 Provider / 角色包 / 世界模组可被安全加载

## 🌏 v1.0（长期：成熟的"AI 平权"基础设施）

> 这一版不追求发版日期，追求"稳定到每个人都可以放心把自己的世界放在上面"。

- [ ] 完整的隐私 & 安全审计（数据从未出过你的实例这件事可被证明）
- [ ] 多人协作世界（可选）：你可以邀请真实朋友进你的世界做客
- [ ] 企业 / 团队版：把"世界"用作团队陪练 / 决策复盘 / 入职导师
- [ ] 移动端公开分发
- [ ] 开发者文档全面化 + Provider SDK

## 💡 不在路线图上、但欢迎讨论的

有些方向我们还在观望，不想承诺、也不想提前否定。欢迎来 Discussions 聊：

- VR / AR 入口
- 多模态主动行为（角色主动发语音、发图、发视频）
- 你在现实里的"地理位置"如何更深地融入世界
- 角色之间的"代际"——老一批居民把故事传给新一批
