# iOS 壳四语种切换支持计划

## Summary

- 目标：iOS 壳支持简体中文、英文、日语、韩语；首次启动无手动偏好时跟随 iOS 系统语言，用户在 App 内“我 -> 设置 -> 语言”手动选择后覆盖系统语言并保存在当前设备。
- 覆盖两层：Web 业务界面即时切换；iOS 原生可见文案按系统语言本地化，包括 App 名称、权限弹窗文案、`YinjiePublicAppName`。
- 执行阶段先保存计划到 `.Codex/plans/ios-shell-i18n-2026-04-27.md`，完成后按项目规则做一次阶段性提交。

## Key Changes

- 扩展 i18n 初始化契约：`AppLocaleProvider` 新增可选系统语言候选列表，初始化优先级固定为 `?locale=` -> 已持久化手动偏好 -> iOS 系统语言 -> `zh-CN`。
- 扩展 locale 解析：支持 iOS 常见语言码如 `zh-Hans-CN`、`zh_CN`、`en-GB`、`ja`、`ko`，统一映射到现有 `zh-CN` / `en-US` / `ja-JP` / `ko-KR`。
- 扩展 iOS runtime：`YinjieRuntime.getConfig()` 返回 `preferredLocales`，来源为 `Locale.preferredLanguages`，Web 启动前通过 `hydrateNativeRuntimeConfig()` 注入。
- 添加 iOS 原生本地化资源：在 `apps/ios-shell/ios/App/App/{zh-Hans,en,ja,ko}.lproj/InfoPlist.strings` 中维护：
  - `CFBundleDisplayName`
  - `YinjiePublicAppName`
  - `NSCameraUsageDescription`
  - `NSPhotoLibraryUsageDescription`
  - `NSPhotoLibraryAddUsageDescription`
  - `NSMicrophoneUsageDescription`
- 更新 iOS 工程守护脚本：
  - `configure-ios-project.mjs` 生成/同步四个 `.lproj/InfoPlist.strings`，并把 `InfoPlist.strings` 加入 Xcode resources 与 `knownRegions`。
  - `doctor-ios.mjs` 检查四语种资源、必需 key、Xcode project 资源引用、runtime plugin 的 `preferredLocales` 输出。
- 更新文档：`apps/ios-shell/README.md` 说明系统语言默认、App 内手动切换、原生权限文案跟随 iOS 系统/App 语言，不受 Web 手动切换即时改写。
- 修正 `LanguageSwitcher` 默认说明文案，使其与实际持久化行为一致：语言偏好保存在当前设备，而不是“只影响当前页面会话”。

## Public Interfaces

- `AppRuntimeConfig` 新增可选字段：
  - `preferredLocales?: string[]`
- `YinjieRuntime.getConfig()` 返回值新增：
  - `preferredLocales: string[]`
- `AppLocaleProvider` 新增可选 prop：
  - `preferredLocales?: readonly string[]`
- 不新增后端 API，不改数据库，不改路由。

## Test Plan

- `pnpm --filter @yinjie/i18n typecheck`
- `pnpm --filter @yinjie/app typecheck`
- `pnpm --filter @yinjie/ios-shell run doctor`
- 如调整 Lingui 文案：执行 `pnpm i18n:compile`
- 手动验收场景：
  - 清空 `localStorage["yinjie-i18n-locale:app"]` 后，模拟/真机 iOS 系统语言为 English / 日本語 / 한국어 / 简体中文时，首次进入 App 显示对应语言。
  - 在“我 -> 设置 -> 语言”切换四种语言，底部 Tab、设置页标题、时间/数字格式随切换刷新，刷新 App 后保持手动选择。
  - iOS 权限弹窗和 App 显示名按系统/App 语言读取 `InfoPlist.strings`。
  - `pnpm ios:configure` 后重复运行不产生无意义抖动。

## Assumptions

- “中文”按现有项目语种定义为简体中文 `zh-CN`，iOS 原生目录使用 `zh-Hans.lproj`。
- 品牌名英文、日文、韩文统一显示为 `Yinjie`；中文显示为 `隐界`。
- Web 内手动语言切换即时生效；iOS 系统权限弹窗文案由系统语言决定，不能被 Web 内 localStorage 语言即时覆盖。
